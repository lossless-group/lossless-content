## Purpose
The [[projects/Augment-It/Specs/apps-microfrontends/ResponseReviewer|ResponseReviewer]] [[Microfrontend Architecture|Microfrontend]] will collect the core content and metadata from API calls made to [[Large Language Models|LLMs]] or [[AI Native Applications|AI Native]] [[Web Scraping|Web Scrapers]]. 

The user will be able to make highlights within the content of the API responses, which will then be saved as the more valuable information generated by the AI Models and Web Scrapers.
